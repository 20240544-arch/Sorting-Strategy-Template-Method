# Sorting System (Design Patterns Lab)

## Description
This project demonstrates the combined use of two behavioral design patterns:
1. **Strategy Pattern**: To swap sorting algorithms (BubbleSort, QuickSort) at runtime.
2. **Template Method Pattern**: To define a fixed file-processing workflow while allowing flexible sorting strategies.
